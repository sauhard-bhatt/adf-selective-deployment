\# ADF Selective Deployment



This repository demonstrates a selective deployment approach for Azure Data Factory.



\## Problem

ADF deployments are state-based, which makes parallel feature releases difficult.



\## Solution

This utility helps identify and deploy only the required artifacts, rather than promoting the full state.



\## Repository Contents

\- `scripts/` - Python utility

\- `adf-sample/` - sanitized sample ADF artifacts

\- `config/` - sample configuration

\- `docs/` - supporting documentation



\## Disclaimer

This is a sanitized reference implementation. All sensitive values and environment-specific identifiers have been removed or replaced with placeholders.
